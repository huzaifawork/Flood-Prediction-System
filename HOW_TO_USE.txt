TROUBLESHOOTING:
===============

Problem: "Python is not installed"
Solution: Install Python and make sure "Add to PATH" is checked

Problem: "Node.js is not installed" 
Solution: Install Node.js LTS version from nodejs.org

Problem: "Setup not completed"
Solution: Run SETUP.bat as administrator first

Problem: Browser doesn't open
Solution: Manually go to http://localhost:5173
